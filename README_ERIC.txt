======================================
 ZwickRoell Order Tracker - Setup Guide
======================================

FIRST TIME ONLY
---------------
1. Copy the "OrderTracker" folder anywhere on your computer
   (for example: C:\OrderTracker  or your Desktop)

2. Double-click "install.bat"
   - This installs Python and all required software automatically.
   - It may take 1-2 minutes. Just wait until you see "Setup complete!".
   - You only need to do this once.

EVERY DAY
---------
3. Double-click "start.bat"
   - A black window will appear briefly while the app starts.
   - Your browser will open automatically to the Order Tracker dashboard.
   - Keep the "Order Tracker Server" window open while you use the app.
     Close it when you are done.

FIRST-TIME CONFIGURATION
-------------------------
- When the dashboard opens, click the settings icon (top right) and
  set the "Root Folder" to the folder that contains all your order folders
  (the folder with the Checklist .docx files inside subfolders).
- Click "Save", then click "Scan Orders" to load all orders.

TROUBLESHOOTING
---------------
- If the browser does not open, go to: http://localhost:8000
- If you see an error about the root folder, make sure the path in
  settings points to your OneDrive orders folder.
- If install.bat says it could not download Python, make sure you have
  an internet connection and try again.

Need help? Contact the person who set this up for you.
